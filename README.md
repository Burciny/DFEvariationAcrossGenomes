The files in the repository are arranged as follows:

Population_data/

Final filtered population genetics data for mouse, fruit fly, and yeast.

GeneBayes_in/ and GeneBayes_out/

Input and output files from the GeneBayes analysis.

Intermediate_data/

Data generated by functions in Scripts/functions/ and used for the final analysis and visualization in the scripts under Scripts/manuscript/.

Scripts/functions/

Functions for all analyses, including creation of intermediate data and calculation of the statistics presented in the paper.
